## 一、比賽定位

### 1.1 所屬比賽
2026 理大人工智能 × 數位人文獎（香港理工大學人文學院主辦）

### 1.2 參賽組別
**B組 —— 客製化AI工具、AI Agent或AI Agent Workflow**

B組定位為「建立AI tutor、文本分析工具、資料抽取工具、知識圖譜、資料集、檢索系統或AI Agent Workflow原型」。本作品以歷史文獻結構化處理為起點，構建一套從原始史料到互動視覺化呈現的完整工作流程，涵蓋文本資料抽取、分類標註、地理編碼與互動地圖生成，屬於B組中「資料抽取工具 + AI Agent Workflow原型」的交叉領域。

### 1.3 主題回應
主題「AI & Classics」要求作品圍繞古典文獻與文化，並清楚說明其與經典或古典學習的關聯。本作品以《明太祖實錄》中洪武年間蕃國朝貢記錄為核心史料，運用AI輔助的數據抽取和視覺化技術，將傳統文獻學研究轉化為可互動、可探索的數位工具。

## 二、數據來源與學術背景

### 2.1 核心數據
附錄二「洪武時期蕃國朝貢一覽表」，已整理為結構化表格，來源為《明太祖實錄》。

### 2.2 數據規模（經精確統計）

| 指標 | 數值 |
|------|------|
| 總朝貢記錄 | 290條 |
| 時間跨度 | 洪武二年（1369）至洪武三十一年（1398），共30年 |
| 獨立藩國名稱 | 26個（經規範化後為20個政治實體） |
| 每條記錄包含 | 時間、藩國、國王、使者、目的、貢物、給賜、史料來源 |

### 2.3 藩國分布與朝貢頻率

**第一梯隊（>30條記錄）：** 高丽/朝鲜（合計82條）、暹罗（42條）、安南（34條）、琉球中山国（31條）

**第二梯隊（10–30條）：** 占城（27條）、朝鲜（含高丽过渡16條）、真腊（12條）、琉球山南国（12條）、爪哇（11條）、琉球山北国（11條）

**第三梯隊（<10條）：** 日本（9條）、三佛齐（6條）

**單次朝貢國（各1條）：** 西洋、浡泥、琐里、拂菻国、览邦、淡巴、阇婆、湓亨、百花、须文那达、爪哇东番、爪哇西番

### 2.4 數據特殊情況

需要在數據處理中特別注意的邊界情況：

1. **高麗→朝鮮過渡**：洪武二十五年（1392）李成桂政變，同年末請更國號，洪武二十六年正式使用「朝鮮」。數據中出現「高丽」「朝鲜」「高丽（朝鲜）」三種寫法，須在可視化中處理為同一地理位置但標注政權更替。

2. **琉球三國**：琉球中山国、琉球山南国、琉球山北国為三個獨立政治實體，分佈於沖繩群島不同區域，須各自標注，不可合併。

3. **爪哇東番/西番**：洪武十二年僅出現一次，為爪哇內部勢力分裂，可在地圖上共用爪哇位置但附加說明。

4. **「琉球」泛稱**：洪武十八年出現一條未區分三國的「琉球」記錄，應歸入琉球中山国（最常遣使者）。

5. **閏月**：部分記錄包含閏月（如「洪武六年閏十一月」「洪武二十三年閏四月」），需正確處理排序。

### 2.5 學術意義

原始數據附有學術按語，引用全海宗《清代韓中朝貢關係考》指出：外國使節來華的任務往往是複雜而非單一的，在賀聖節的同時也可謝恩、奏聞，在陳奏時也可帶同貢物。本作品的互動地圖正是為了呈現這種「歷史真實的複雜性」，讓使用者從時間、空間、物質交換三個維度探索朝貢關係。

## 三、作品概述

### 3.1 產品定義

一個基於Web的互動式歷史地圖工具，以明洪武年間東亞/東南亞底圖為基礎，讓使用者通過時間滑桿操控，觀察各藩國的朝貢時間分布、貢物類型、路線軌跡與使團行進動畫。

### 3.2 核心交互流程

```
用戶拖動時間滑桿（洪武2年 → 31年）
    ↓
該年朝貢的藩國在地圖上點亮（發光圓點）
    ↓
顯示從藩國到南京（應天府）的朝貢路線（弧線軌跡）
    ↓
路線上出現該國主要貢物的微縮圖標
    ↓
如用戶靜默3秒 → 使團車馬沿軌跡向南京行進的動畫
    ↓
用戶切換時間 → 動畫停止，更新為新年份數據
    ↓
用戶點擊藩國節點 → 跳轉到該國專屬子頁面（歷年朝貢詳情）
```

### 4.2 人工判斷範圍

以下環節涉及歷史學判斷，由團隊成員人工完成：

- 藩國地理位置的最終確認（如「西洋」「琐里」「拂菻」等古地名的現代對應）
- 高麗→朝鮮政權過渡的呈現方式
- 貢物分類的學術合理性
- 路線設計（海路/陸路）的歷史依據
- 所有文字說明與學術標注
